# Credit Card Fraud Detection

A machine learning project to detect fraudulent credit card transactions using classification algorithms.

## About This Project

This project uses machine learning classification algorithms to identify fraudulent credit card transactions. The model is trained on real-world transaction data to achieve high accuracy in fraud detection.

## Dataset

The dataset used for this project is available on Kaggle:

**[Credit Card Fraud Detection Dataset - Kaggle]("/content/creditcard.csv.zip")**

### How to Use the Dataset:
1. Visit the Kaggle link above
2. Download the `creditcard.csv` file
3. Place it in the same folder as the notebook
4. Run the notebook

## Technologies Used

- Python
- Pandas (Data Analysis)
- NumPy (Numerical Computing)
- Scikit-learn (Machine Learning)
- Matplotlib & Seaborn (Visualization)
